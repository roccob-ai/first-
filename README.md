# DealRoom

DealRoom is a modern Next.js finance dashboard that helps students understand
major M&A deals through an investment banking lens.

## Features

- Dashboard with headline metrics, search, industry filter, and deal cards
- Deal detail pages with strategy, synergies, financing, risks, valuation, and
  interview questions
- Industries page with sector-level deal value summaries
- About page explaining how students can use the product for recruiting prep
- Mock data only for now, organized in `lib/deals.ts`

## Tech stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- lucide-react icons

## Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the app.
